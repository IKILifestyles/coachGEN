# CoachGEN Landing Page

Website Landing Page dành cho nền tảng coaching dựa trên triết lý **DNA – Thế hệ kế thừa – Phát triển tiềm năng cá nhân**.

## 🚀 Công nghệ sử dụng
- **HTML5** (Semantic structure)
- **CSS3** (Vanilla, CSS Variables, Grid/Flexbox, Animations)
- **JavaScript** (Thuần, DOM Event Listeners, Intersection Observer)
- **Lucide Icons** (CDN trực tiếp)

## 💻 Cài đặt và Khởi chạy
Dự án được xây dựng hoàn toàn bằng các công nghệ chuẩn Web (Web Native) nhằm đảm bảo tốc độ và sự đơn giản, bạn không cần phải cài đặt NodeJS hay một Framework / Build tool nào. 

### Chỉ cần làm theo 1 trong 2 cách sau:
**Cách 1: Chạy trực tiếp trên trình duyệt**
- Click đúp vào file `index.html` trong thư mục `CoachGEN landing page`. Web sẽ mở ngay lập tức trên trình duyệt của bạn (Chrome, Safari, Edge...).

**Cách 2: Chạy qua Live Server (Dành cho Developer)**
- Mở thư mục dự án bằng VS Code.
- Cài Extension **Live Server** (tìm kiếm trong kho VS Code extensions).
- Click chuột phải vào `index.html` chọn **"Open with Live Server"**.

## ✨ Tính năng nổi bật
1. **Thiết kế UI/UX hiện đại**: Tông màu Gradient được định nghĩa chuyên nghiệp (Trust Blue, Insight Blue, Insight Purple) tạo cảm giác kiến thức chuyên sâu và hướng tới tương lai.
2. **Hiệu ứng Scroll Reveal**: Các phần tử nhẹ nhàng lướt lên mượt mà dựa trên trạng thái cuộn trang (sử dụng `IntersectionObserver` tối ưu hóa thay vì listener scroll truyền thống).
3. **Mô phỏng DNA 3D**: Animation lưới (Mesh glow) đằng sau khu vực mở đầu (Hero banner) chậm rãi chuyển động không ngừng mô phỏng chuỗi gen tiềm năng.
4. **Responsive tuyệt đối**: Sử dụng Flexbox & CSS Grid với các điểm ngắt breakpoint Media Query tiêu chuẩn đảm bảo mọi thiết bị (từ Desktop to nấc nhỏ nhất như iPhone SE) cũng đều hiển thị xuất sắc. Menu có định dạng Hamburger gài vuốt Mobile-friendly.
5. **Form xử lý mô phỏng**: Chặn tải lại hành vi mặc định và in toàn bộ JSON Data của Contact xuống `console.log()` kèm thông báo Alert cho phép nghiệm thu UI/UX Form.
